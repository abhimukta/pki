# PKI Toolkit
Purpose: Free, vendor-neutral browser tools for PKI practitioners.
Constraints:
- Pure client-side HTML/JS. No backend, no data leaves the browser
  (critical: users paste certificates and CSRs — privacy is the product).
- Single-file HTML pages where possible, deployable to Cloudflare Pages.
- Executive navy/gold/cream visual style, consistent with abhi's site.
- Plain-English explanations for non-technical readers on every tool page.
- No employer references. Company-independent by design.
